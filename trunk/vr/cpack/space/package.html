Space Simulation

This simulation contains the sun, nine planets, some of their moons and ~90k stars. This is all rendered in proportion to real distances and times and is navigable by keyboard and mouse.

This package is the first test of a design goal to separate scene content from the rest of the architecture needed to render and exist in a 3D scene. Where possible, code from org.freality.gui.three.* has been used for any such generic scene functionality.

Until version 1.0, the intended audience for this is java developers.

To run, simply run "java -jar [the version you download]". This will load the datasets remotely, which is fine for a simple demo, but very slow given my link speed. So until I add caching by default you may want to download the datasets and put them in your classpath. Which brings us to the todo...

History

Current Dev - Fixed moons having same inclination as parent planet. Atmospheres.

2004-12-30: 0.3 - First major debugging, cleanup and package management since 0.1.. Distances and times to scale. HYG database of stars added.

2004-10-01: 0.2 - After a long pause in development, delayed mainly be confusion in code structure, code was reorganized into a main package with sub-packages for data and ontological model. Eliptical orbits used and properly oriented around the Sun/parent body. Space-ship-ish navigation added.

2002-08-15: 0.1 - Initial version was inspired by Celestia and was a first cut at using Java3D. Included the Sun, nine planets, some of their moons and random stars. Wasn't to scale; orbits were circles centered on the Sun. The code wasn't very well split into "generic" and "space" modules.
